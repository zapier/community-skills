# community-skills

Community-contributed agent skills you can install and use. Zapier hosts this repository; each skill is contributed and licensed by its author.

## Using a skill

Install a skill into your agent with the skills CLI:

```bash
npx skills add zapier/community-skills --skill <skill-name>
```

This drops the skill into your local agent configuration (e.g. `.claude/skills/` or `.cursor/skills/`). Skills are also indexed on [skills.sh](https://skills.sh).
